Introduction

Input handling for sokol_app

Discussion

Provides an input manager for sokol_app, handling keyboard, mouse, and gamepad input.

Macro Definitions

SOKOL_GAMEPAD_MAX

The maximum number of gamepads supported.

SOKOL_INPUT_NO_GAMEPADS

Disable gamepad support.

SOKOL_GAMEPAD_MAX

The maximum number of gamepads supported. 

#define SOKOL_GAMEPAD_MAX 1
Discussion

This defines the maximum number of gamepads that can be connected at the same time. The default value is 1, but it can be changed to support more gamepads.

SOKOL_INPUT_NO_GAMEPADS

Disable gamepad support. 

#define SOKOL_INPUT_NO_GAMEPADS
Discussion

This macro can be defined to disable gamepad support in the sokol_input.h header. It is useful for platforms where gamepad support is not available or not needed. This is automatically defined for Android, Emscripten, and iOS platforms.
